What is GeoAuth?

GeoAuth is Sensewhere's unique transaction fraud detection service.

How does it work?

GeoAuth validates a transaction by matching its WiFi fingerprint against the WiFi profiles of vendors.

WiFi Signatures

Phone

A smartphone picks up an unique set of signals from nearby WiFi Access Points.

Vendor

Likewise, each vendor has an unique list of WiFi Access Points.

Do their signatures match?

If the transaction is made at the vendor, the phone should have picked up some of the MAC addresses from the Vendor WiFi Access Points.

Algorithm

Measuring similarity

How similar is the list of MAC addresses picked up by the transaction phone to the list of vendor MACs?

Jaccard Similarity

Given two sets A and B, the Jaccard index

\[\begin{aligned} J(A, B) = \frac{|A \cap B |}{|A \cup B|} \end{aligned}, \]

is a common measure of similarity in data mining.

In the previous case:
A = Transaction, B = Vendor, Jaccard Similarity = 7/15.

Not very alike!

GeoAuth Similarity

\[\begin{aligned} \mathbf{GeoAuthSim}(A, B) = \frac{|A \cap B|}{\min(|A|, |B|)} \end{aligned}\]

Now GeoAuth Similarity = 7/10.
